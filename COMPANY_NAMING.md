# 🏢 회사명 자동 추출 가이드

파일명에서 회사명을 자동으로 추출하는 기능입니다.

---

## 📝 파일명 작성 규칙

### 기본 형식
```
[회사명]_파일명.pdf
```

### 예시
```
✅ 넥슨_포트폴리오.pdf → 회사: 넥슨
✅ 엔씨소프트_시스템기획서.pdf → 회사: 엔씨소프트
✅ 크래프톤_레벨디자인.pdf → 회사: 크래프톤
✅ 넷마블-게임기획.pdf → 회사: 넷마블
✅ [펄어비스] 포트폴리오.pdf → 회사: 펄어비스
✅ 넥슨_엔씨소프트_합격.pdf → 회사: 넥슨, 엔씨소프트 (복수 가능)
```

---

## 🎯 지원하는 구분자

다음 구분자들을 인식합니다:
- `_` (언더스코어) ← **추천!**
- `-` (하이픈)
- ` ` (공백)
- `[` `]` (대괄호)
- `(` `)` (괄호)

## 💡 사용 예시

### Case 1: 단일 회사
```
파일명: 넥슨_게임기획_포트폴리오.pdf
추출: ["넥슨"]
```

### Case 2: 복수 회사 (여러 회사 합격)
```
파일명: 넥슨_엔씨소프트_크래프톤_합격.pdf
추출: ["넥슨", "엔씨소프트", "크래프톤"]
```

### Case 3: 영문 표기
```
파일명: NEXON_Portfolio.pdf
추출: ["넥슨"] (자동으로 한글로 정규화)
```

### Case 4: 대괄호 사용
```
파일명: [넥슨] 시스템기획서 v2.pdf
추출: ["넥슨"]
```

### Case 5: 일반게임회사 (일반)
```
파일명: 일반게임회사_포트폴리오.pdf
추출: ["일반게임회사"]

파일명: 게임회사_기획서.pdf
추출: ["일반게임회사"] (자동으로 정규화)

파일명: 게임사_레벨디자인.pdf
추출: ["일반게임회사"] (자동으로 정규화)
```

### Case 6: 회사명 없음
```
파일명: 포트폴리오.pdf
추출: [] (빈 배열)
```

## 🎨 업로드 시 자동 처리

### 1. Training 페이지에서 업로드
```
1. 파일 선택: "넥슨_포트폴리오.pdf"
2. 업로드 시작
3. 자동으로 "넥슨" 추출
4. DB에 저장: companies: ["넥슨"]
5. 완료 메시지: "완료 (넥슨)"
```

### 2. Data 페이지에서 확인
```
- 파일명: 넥슨_포트폴리오.pdf
- 회사: 넥슨 (태그로 표시)
```

## ⚠️ 주의사항

### 1. 정확한 회사명 사용
```
✅ 넥슨_포트폴리오.pdf → 인식됨
✅ NEXON_portfolio.pdf → 인식됨 (영문도 가능)
❌ 넥슨코리아_포트폴리오.pdf → 인식 안 됨 (정확히 "넥슨")
❌ NC_포트폴리오.pdf → "엔씨소프트"로 인식
```

### 2. 띄어쓰기 주의
```
✅ 넥슨_포트폴리오.pdf
✅ 넥슨-포트폴리오.pdf
⚠️ 넥 슨_포트폴리오.pdf → 인식 안 될 수 있음
```

### 3. 회사명은 맨 앞에
```
✅ 넥슨_포트폴리오.pdf (추천!)
✅ [넥슨] 포트폴리오.pdf
⚠️ 포트폴리오_넥슨.pdf (가능하지만 맨 앞 추천)
```

## 🔧 회사명 추가 요청

리스트에 없는 회사를 추가하고 싶다면:
1. `lib/company-parser.ts` 파일 열기
2. `GAME_COMPANIES` 배열에 회사명 추가
3. `COMPANY_ALIASES` 객체에 정규화 규칙 추가

**또는** 개발자에게 요청하세요!

## 📊 통계 활용

회사별로 자동 분류되므로:
- 회사별 평균 점수 확인 가능
- 회사별 합격 포트폴리오 개수 확인
- 특정 회사 합격자 포트폴리오만 필터링

## 💡 Pro Tip

### 파일명 작성 팁
```

- 넥슨_시스템기획_홍길동.pdf
```
