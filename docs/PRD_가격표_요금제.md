## 2. 회차권 (크레딧)

### 2.1 크레딧 패키지

| 패키지 | package_type | 가격 | 회당 단가 | 할인율 | 비고 |
|--------|-------------|------|-----------|--------|------|
| **무료** | - | 0원 | - | - | 회원가입 시 1회 자동 지급 |
| **1회권** | `credit_1` | 2,900원 | 2,900원 | - | 기본 단가 |
| **5회권** | `credit_5` | 7,900원 | 1,580원 | 45% | - |
| **10회권** | `credit_10` | 12,900원 | 1,290원 | 55% | 월 구독보다 1,000원 저렴 |

### 2.2 크레딧 규칙
- 크레딧 만료 없음 (영구 유효)
- 분석 1회 = 크레딧 1개 소모
- 구독자는 크레딧 차감 없음 (무제한)
- 회원가입 시 `analysis_credits = 1` (첫 1회 무료)

## 3. 구독 요금제 (무제한)

### 3.1 플랜 비교표

| 플랜 | 가격 | 기간 | 분석 횟수 | 프로젝트 | AI 모델 | 비고 |
|------|------|------|-----------|----------|---------|------|
| **월 무제한** | 13,900원/월 | 1개월 | 무제한 | 무제한 | Claude Sonnet | 추천 플랜 |
| **3개월 무제한** | 39,000원 | 3개월 | 무제한 | 무제한 | Claude **Opus** | 프리미엄 AI |
| **컨설팅 포함** | 컨설팅 결제 시 | 1개월 | 무제한 | 무제한 | Claude Sonnet | 자동 부여 |

### 3.2 할인 정책

| 할인 이름 | 대상 플랜 | 정가 | 할인가 | 할인액 | 조건 |
|-----------|-----------|------|--------|--------|------|
| 게임캔버스 수강생 | 월 무제한 | 13,900원 | **5,900원** | 8,000원 | 할인코드 입력 |

### 3.3 할인코드 관리
- 환경변수: `GAMECANVAS_DISCOUNT_CODES` (쉼표 구분)
- 검증: `app/actions/payment.ts` → `validateGamecanvasCode()`
- 적용 대상: **monthly 플랜만** (three_month은 대상 아님)
- 할인 적용 시 주문명: "디자이닛 월 구독 (게임캔버스)"

## 6. 가격 변경 절차

```
1. 이 문서(PRD_가격표_요금제.md) 먼저 수정
2. app/actions/payment.ts의 CREDIT_PRICES / SUBSCRIPTION_PRICES 수정 (서버 기준값)
3. 위 5.1~5.4 표의 모든 파일에서 UI 가격 수정
4. 테스트: 결제 페이지에서 실제 금액 확인
5. 커밋 & 배포
```

> ⚠️ **주의**: 서버(payment.ts)와 UI 가격이 다르면 결제 오류 발생.
> 컨설팅은 서버에서 DB 금액을 재확인하므로, DB에 이미 생성된 주문의 금액은 수정 불가.

## 7. 가격 표시 규칙

| 규칙 | 예시 |
|------|------|
| 천 단위 쉼표 | 17,900원 (O) / 17900원 (X) |
| 할인 시 원가 취소선 | ~~53,700원~~ → 49,000원 |
| 할인율 뱃지 | "10% 할인" 뱃지 표시 |
| 월/기간 표기 | "원 / 월", "원 / 3개월" |
| 무료는 숫자 | "0원" (O) / "무료" → 별도 문구로 표시 |

## 8. 알려진 이슈

| 항목 | 설명 |
