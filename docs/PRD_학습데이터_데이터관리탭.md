# PRD: 학습 데이터 관리 탭

## 개요
- **라우트**: `/admin/training` (activeTab = "data")
- **접근**: 관리자 전용
- **목적**: 학습된 포트폴리오 데이터 조회, 검수, 정리

## 역할 정의
**"데이터 조회 + 관리 + 정리"** — 이미 학습된 데이터를 확인하고 관리하는 작업

### 포함 기능
1. **전체 통계**: 총 학습 데이터 수, 선택 수, 평균 점수
2. **데이터 목록**: 포트폴리오별 파일명, 회사, 점수, 연도 표시
3. **개별 삭제**: 각 포트폴리오 삭제 버튼
4. **일괄 삭제**: 체크박스 선택 → 선택 삭제
5. **회사 재분류**: 파일명 기준으로 회사명 일괄 재파싱

### 포함하지 않는 기능
- 파일 업로드 → 업로드 탭
- Gemini/Claude AI 분석 실행 → 업로드 탭
- 벡터 재임베딩 → 업로드 탭
- 심층 분석 → 업로드 탭

## UI 구성

### 1. 전체 통계 카드
- 총 학습 데이터 수
- 선택된 수
- 평균 점수

### 2. 학습 데이터 목록 카드
- **헤더**: "학습 데이터 목록" + 회사 재분류 버튼 + 선택 삭제 버튼
- **전체 선택 체크박스**
- **목록** (스크롤 가능, 최대 600px):
  - 체크박스
  - 파일명
  - 회사 태그 (파란색 뱃지)
  - 연도
  - 점수
  - 삭제 버튼 (개별)
- **안내 문구**: "학습 데이터를 삭제하면 AI가 해당 데이터를 더 이상 참고하지 않습니다."

## 서버 액션 (admin.ts)
- `getPortfolioList()` — 전체 데이터 조회
- `deletePortfolio()` — 개별 삭제
- `deleteMultiplePortfolios()` — 일괄 삭제
- `reclassifyAllCompanies()` — 회사명 재분류
- `getCompanyStats()` — 회사별 통계

## 주의사항
- 삭제 시 confirm 다이얼로그 필수
- 삭제하면 portfolio_chunks, portfolio_analysis도 cascade 삭제됨
- 회사 재분류는 파일명 기반 파싱이므로 파일명 규칙이 중요
