# PRD: 학습 데이터 관리 탭

## 개요
- **라우트**: `/admin/training` (activeTab = "data")
- **접근**: 관리자 전용
- **목적**: 학습된 포트폴리오 데이터 조회, 검수, 정리 + 심층분석 결과 확인 + 공통점 관리

## 역할 정의
**"데이터 조회 + 관리 + 분석 결과 확인"** — 이미 학습된 데이터를 확인하고 관리하며, 심층 분석 결과와 공통점을 열람하는 작업

### 포함 기능
1. **전체 통계**: 총 학습 데이터 수, 선택 수, 평균 점수
2. **데이터 목록**: 포트폴리오별 파일명, 회사, 점수, 연도 표시
3. **심층 분석 결과 펼치기**: 각 포트폴리오 행 클릭 시 15개 카테고리 점수 + 강점/약점/요약 표시
4. **개별 삭제**: 각 포트폴리오 삭제 버튼
5. **일괄 삭제**: 체크박스 선택 → 선택 삭제
6. **회사 재분류**: 파일명 기준으로 회사명 일괄 재파싱
7. **합격자 공통점 관리**: 공통점 추출 + 카테고리별 열람 (기존 `/admin/success-patterns` 통합)

### 포함하지 않는 기능
- 파일 업로드 → 업로드 탭
- Gemini/Claude AI 분석 실행 → 업로드 탭
- 벡터 재임베딩 → 업로드 탭
- 심층 분석 실행 → 업로드 탭

## UI 구성

### 1. 전체 통계 카드
- 총 학습 데이터 수
- 선택된 수
- 평균 점수

### 2. 학습 데이터 목록 카드
- **헤더**: "학습 데이터 목록" + 회사 재분류 버튼 + 선택 삭제 버튼
- **심층분석 통계 바**: 분석 완료 수 / 전체 수
- **전체 선택 체크박스**
- **목록** (스크롤 가능, 최대 600px):
  - 체크박스
  - 펼치기 토글 (화살표, 심층분석 완료 시 보라색)
  - 파일명
  - 회사 태그 (파란색 뱃지)
  - 심층분석 뱃지 (보라색, 분석 완료 시)
  - 연도
  - 점수
  - 삭제 버튼 (개별)
- **펼침 패널** (행 클릭 시):
  - 15개 카테고리 점수 (5열 그리드, 점수별 색상)
  - 요약 텍스트
  - 강점/약점 (2열 그리드)
  - 핵심 특징 키워드 (보라색 뱃지)
  - 미분석 시 안내 메시지
- **안내 문구**: "학습 데이터를 삭제하면 AI가 해당 데이터를 더 이상 참고하지 않습니다."

### 3. 합격자 공통점 카드
- **헤더**: "합격자 공통점" + 공통점 추출 버튼
- **카테고리 탭**: 일반 공통점 + 회사별 탭
- **패턴 목록** (스크롤 가능, 최대 500px):
  - 번호 (원형 뱃지)
  - 제목
  - 중요도 뱃지 (핵심/유용/참고)
  - 설명
  - 예시 파일
- **하단 통계**: 총 패턴 수 | 일반 | 회사별

## 서버 액션 (admin.ts)
- `getPortfolioList()` — 전체 데이터 조회
- `getPortfolioAnalysisAll()` — 심층 분석 전체 데이터 조회
- `deletePortfolio()` — 개별 삭제
- `deleteMultiplePortfolios()` — 일괄 삭제
- `reclassifyAllCompanies()` — 회사명 재분류
- `getCompanyStats()` — 회사별 통계
- `getSuccessPatterns()` — 공통점 조회
- `extractSuccessPatterns()` — 공통점 추출 (배치 분할)

## 주의사항
- 삭제 시 confirm 다이얼로그 필수
- 삭제하면 portfolio_chunks, portfolio_analysis도 cascade 삭제됨
- 회사 재분류는 파일명 기반 파싱이므로 파일명 규칙이 중요
- 공통점 추출은 기존 데이터를 삭제하고 재생성 (3~4분 소요)
- 심층 분석 결과는 portfolio_analysis 테이블에서 조회 (실행은 업로드 탭)
