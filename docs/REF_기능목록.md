## 1. 핵심 기능

### 1-1. 사용자 문서 분석
- **URL 분석** (`analyzeUrlDirect`): 웹 페이지 URL 입력 → 텍스트 추출 → Claude AI 분석
- **파일 분석** (`analyzeDocumentDirect`): PDF/문서 업로드 → Claude AI 분석
- **분석 결과**: 총점 + 15개 카테고리별 점수 + 상세 피드백
  - 기본 5개: 논리력, 구체성, 가독성, 기술이해, 창의성
  - 게임디자인 10개: 코어메카닉, 밸런스/경제, 레벨/맵, UX/UI, 시스템기획 등
- **벡터 서치**: 유사 합격자 포트폴리오의 실제 텍스트를 AI에 제공하여 구체적 비교 피드백
- **AI 엔진**: Anthropic Claude (Sonnet/Opus)

### 1-2. 합격자 포트폴리오 관리 (관리자)
- PDF/엑셀/CSV 파일 업로드 → AI 분석 → 벡터 임베딩
- **AI 엔진**: Google Gemini 2.0 Flash (학습 데이터 분석)
- 현재 **187개** 합격자 포트폴리오 학습 완료 (204 청크)
- 합격자 공통점 50가지 자동 추출 (배치 분할 방식)
- **NotebookLM**: 187개 포트폴리오를 소스로 업로드하여 직접 질의 (회사별 특징, 평균 점수, 공통 패턴 등 추출)

### 1-3. 결제 시스템
- TossPayments 연동 (일반결제 + 빌링키 자동결제)
- 크레딧(회차권): 1회(2,900원), 5회(7,900원), 10회(12,900원)
- 구독: 월 무제한(13,900원), 3개월 무제한(39,000원)
- 크레딧 우선 소모 정책: 크레딧 > 구독 순서
- 크레딧 셀프 환불: 마이페이지에서 직접 환불 (7일 이내)

### 1-4. 인증/계정
- Supabase Auth (Google, Kakao, Apple 소셜 로그인 + 이메일)
- 마이페이지: 분석 이력 조회, 프로젝트 관리, 구독/크레딧 게이지
- 관리자 이메일 기반 권한 분리
