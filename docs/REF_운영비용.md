## 참고

- **Vercel Pro 업그레이드 이유**: Hobby 플랜은 서버 액션 10초 제한 → 임베딩 처리 시 타임아웃 발생
- **Supabase Pro 업그레이드 이유**: pgvector(벡터 검색) + 스토리지 용량 + DB 성능
- **OpenAI 크레딧 $6.50**: 임베딩 전용. 포트폴리오 수천 개 처리해도 $1 이하. 한참 쓸 수 있음
- **Anthropic API**: 사용자가 분석할 때마다 과금. 1건당 약 $0.01~0.05
- **Gemini API**: 관리자가 합격자 포트폴리오 분석할 때 사용. 현재 무료 티어로 충분
