```typescript
// 사용자 브라우저에서 실행되는 코드
export function createClient() {

    process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY!,
  )

```

### 3. 데이터베이스 유틸리티 함수

#### `lib/supabase/portfolios.ts`

모든 데이터베이스 작업을 쉽게 할 수 있는 함수들:

```typescript
// 포트폴리오 생성
createPortfolio(data)

// 포트폴리오 조회
getPortfolio(id)
getAllPortfolios()
getPortfoliosByCompany(company)

// 통계 조회
getCompanyStats()
getOverallStats()

// 파일 업로드
uploadPortfolioFile(file, fileName)

// 삭제
deletePortfolio(id)
deletePortfolioFile(filePath)
```

---

### 4. 데이터베이스 스키마

#### `scripts/001_create_portfolios.sql`

포트폴리오 저장을 위한 테이블을 자동으로 만들어줍니다:

**테이블**: `portfolios`
- 기본 정보: 파일명, 회사, 연도, 문서 타입
- AI 분석 결과: 5가지 점수 (논리력, 구체성, 가독성, 기술이해, 창의성)
- 요약: 태그, 강점, 보완점
- 원본: 텍스트, 파일 URL

**뷰**: `company_stats` - 회사별 평균 점수
**뷰**: `overall_stats` - 전체 통계

**보안**: RLS 정책 (누구나 읽기, 서버만 쓰기)
