## 🚀 당신이 할 일 (딱 3가지!)

### 1. 환경 변수 파일 만들기 (1분)

프로젝트 폴더에 `.env.local` 파일을 만들고 이 내용을 붙여넣으세요:

```
NEXT_PUBLIC_SUPABASE_URL=https://프로젝트아이디.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=긴_키_문자열
GOOGLE_GENERATIVE_AI_API_KEY=제미나이_키
```

**키는 어디서?**
- Supabase 키: https://supabase.com → 프로젝트 → Settings → API
- Gemini 키: https://aistudio.google.com/app/apikey

---

### 2. Supabase에서 테이블 만들기 (1분)

1. https://supabase.com/dashboard 접속
2. **SQL Editor** 클릭 (왼쪽 메뉴)
3. 이 파일 열기: `scripts/001_create_portfolios.sql`
4. 전체 복사해서 SQL Editor에 붙여넣기
5. **Run** 버튼 클릭

---

### 3. 파일 저장소 만들기 (30초)

1. Supabase에서 **Storage** 클릭 (왼쪽 메뉴)
2. **New bucket** 클릭
3. 이름: `resumes`
4. **Public bucket** 체크 ✓
5. **Create** 클릭

---

## 🎉 끝! 이제 실행하세요

터미널에서:

```bash
npm install    # 처음 한 번만
npm run dev    # 서버 시작
```

브라우저에서 http://localhost:3000 열기

## ❓ 안 되면?

1. `.env.local` 파일이 정말로 프로젝트 **최상위 폴더**에 있나요?
2. Supabase에서 `resumes` 버킷 만들었나요?
3. SQL 스크립트 실행했나요?
4. 서버 재시작: `Ctrl+C` 누르고 다시 `npm run dev`
